About this tool

Everything here runs in your browser. Nothing is uploaded anywhere. "Download TikZ" writes a standalone .tex file matching what you see, drawn with plain tikz-3dplot commands at your current angle. "Download PDF" compiles that file to a vector PDF client-side, using a WebAssembly build of pdfTeX. The first compile fetches LaTeX packages over the network and takes longer.
